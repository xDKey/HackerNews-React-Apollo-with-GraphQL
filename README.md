## This is the Application for studying GraphQL with ReactJS and Appolo Client.

### Quick start: 
```
$ mkdir HackerNews
$ cd HackerNews/
$ git clone https://github.com/xDKey/HackerNews-React-Apollo-with-GraphQL.git
$ npm i
$ npm start

//in new terminal window
$ cd server/
$ npm i
$ npm run dev

```
